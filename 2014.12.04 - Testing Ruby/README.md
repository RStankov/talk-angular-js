Testing Ruby
==================

Demo code for my "Testing Ruby" lecture at [Hack Bulgaria](https://hackbulgaria.com/)'s [Core Ruby](http://ruby.hackbulgaria.com/) course.

Slides can be found at [Speaker Deck](https://speakerdeck.com/rstankov/testing-ruby).


### Installation:

```
gem install bundler
bundler install
```

### Running examples:


```
ruby no_test_1.rb
ruby no_test_2.rb
ruby no_test_3.rb

ruby tdd_1.rb
ruby tdd_2.rb
ruby tdd_3.rb
ruby tdd_4.rb
ruby tdd_5.rb
ruby tdd_6.rb
ruby tdd_7.rb
ruby tdd_8.rb
```

#### Final code:

```
ruby calculation.rb
```
